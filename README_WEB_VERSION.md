# 회계업무 자동화 WEB v1.0

정리일: 2026-05-06

## 현재 개발 기준

- 활성 개발 대상: 회계업무 자동화 WEB v1.0
- 활성 작업 루트: `web_v1`
- 기존 CS 버전은 참고자료로만 유지한다.

## 폴더 구성

- web_v1: WEB v1.0 신규 개발 루트
- manager_server: 기존 CS 버전 참고자료. 담당자용 v6.2 / 서버용 v3.2 소스와 spec
- manager_server/dist: 기존 담당자용 v6.2 실행파일 참고자료
- tax_crawler: 세금계산서 크롤러 최신 구동 파일. WEB v1.0 백엔드에서 재사용 예정
- tax_crawler/docs: 크롤러 문서
- docs: 작업 메모 및 md 문서 모음
- support: 설정, ERP 핸들러, 공통 양식, 핫픽스 파일

## WEB v1.0 방향

- 사용자는 웹 담당자 화면에서 필요한 기능을 선택한다.
- 실제 크롤링, AI 분석, ERP 입력, PDF/프린터 출력은 운영서버에서 수행한다.
- 웹 화면에는 작업 진행률과 로그를 표시한다.
- 작업 완료/실패 시 Chrome/Edge 브라우저 알림을 띄운다.
- 운영서버 GUI 자동화 충돌 방지를 위해 ERP/프린터 작업은 작업 큐에서 순차 실행한다.

## 제외한 것

- backup_* 파일/폴더
- build, __pycache__, 테스트 출력 폴더
- 과거 버전 exe/spec/py
